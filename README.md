## Color conversion formulas
### 1. RGB - HEX
```
R/16 = x1 + y1/16
G/16 = x2 + y2/16
B/16 = x3 + y3/16

e.g.
rgb(29, 31, 32) -> #1d1f20
**29**/16 = **1** + 13(**d**)/16
**31**/16 = **1** + 15(**f**)/16
**32**/16 = **2** + **0**/16

10 11 12 13 14 15
a  b  c  d  e  f
```
