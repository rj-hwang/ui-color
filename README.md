## Color conversion formulas
### 1. RGB - HEX
```
R/16 = x1 + y1/16
G/16 = x2 + y2/16
B/16 = x3 + y3/16

e.g.
rgb(29, 31, 32) -> #1d1f20
<b>29</b>/16 = <b>1</b> + 13(<b>d</b>)/16
<b>31</b>/16 = <b>1</b> + 15(<b>f</b>)/16
<b>32</b>/16 = <b>2</b> + <b>0</b>/16

10 11 12 13 14 15
a  b  c  d  e  f
```
